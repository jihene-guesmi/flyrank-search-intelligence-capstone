# Assignment: Decide Once: Build Your Identity Kit (FL-01)

## 1. Portfolio Typography Frame

- **Heading Font:** `Inter` (Bold / Semi-Bold weight for structural data metrics and titles)
- **Body Font:** `JetBrains Mono` or `Roboto Mono` (A clean, uniform monospace or clean sans-serif layout that ensures code snippets, DuckDB tables, and standard copy read with equal structural alignment)

## 2. Minimalist Architectural Color Palette

To ensure that the engineering data charts and machine learning metrics remain the most visible elements on the screen, we establish a quiet, low-friction palette:

- **Primary Background (Near-White):** `#F8F9FA` (A soft, off-white workspace canvas that eliminates eye strain)
- **Primary Text (Near-Black):** `#1A1D20` (A deep, dark gray that provides high-contrast scannability without the harshness of pure black)
- **Structural Framing Border:** `#DEE2E6` (A light gray line color for clean data tables and markdown partitions)
- **Accent Interactive Indicator:** `#0066CC` (A calm, professional technical blue reserved strictly for code hyper-links and primary Calls to Action)

## 3. Brand Favicon / Logo Monogram

- **Visual Concept:** A minimalist typographic monogram set in the clean heading font.
- **Design Layout:** `[ JG // SI ]` 
- **Meaning:** Represents Jihene Guesmi // Search Intelligence. This acts as a clean, text-based branding element in the navigation header and favicon space.
