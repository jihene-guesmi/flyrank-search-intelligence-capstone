## 1. Case Study 1: FlyRank AI Search Intelligence Capstone

*   **The Problem:** 
    Traditional search engines struggle with keyword-matching limitations. When user search phrases drift linguistically or mimic conversational patterns, standard algorithms fail to align these expressions with static text assets, causing visible keyword coverage drops across multi-million row business portfolios.
*   **What I Did:** 
    I built a two-stage unsupervised ML pipeline utilizing Python. I transformed chaotic raw search sequences into high-dimensional vector representations using dense sentence embeddings (`all-MiniLM-L6-v2`). I engineered semantic clustering mechanisms to isolate structural language gaps, establishing strict out-of-sample data validation filters to eliminate feature dependencies and target leakage during pipeline execution.
*   **What Came Of It:** 
    The resulting system aggregates user queries into contextual, structural intent clusters. This provides content managers with a prioritized, data-driven content action playbook to resolve underlying text misalignments. In testing, the pipeline successfully identified non-obvious intent gaps across an 81.8M row warehouse dataset. 
*   **Next Time / Optimization:** 
    Next time, I will introduce temporal tracking features to evaluate how vector cluster coordinates shift dynamically month-over-month.
